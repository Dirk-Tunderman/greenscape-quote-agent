# Results

Sub-agents write one file per case run here.

**Filename convention:** `<case-id>-<runner>-<YYYYMMDD-HHMM>.md`

Examples:
- `01-runnerA-20260505-1530.md`
- `04-runnerA-20260505-1542.md`
- `02-runnerB-20260505-1545.md`

The structure of each result file is defined in `../SUB-AGENT-INSTRUCTIONS.md` Step 2f.

After all 6 results are written, the main chat aggregates into `../SUMMARY.md`.
